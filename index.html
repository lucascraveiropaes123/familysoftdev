About

Developed by Family SoftDev, this web player is design to be as simples as possible, and can be used and modified only with HTML, CSS and JS. Our goal was to creat the best and simple HTML5 Web Music Player. One that can be used by novice programmers without the need for advanced knowledge in JavaScript, PHP, or even jQuery

How To Use

Getting Started

1 - Add the following code on head of your page

<link href="player/css/style.css" rel="stylesheet"/>
<link href="player/css/landscape.css" rel="stylesheet" media="(max-height: 400px) and (max-width: 700px)">
<link href="player/css/animations.css" rel="stylesheet"/>

2 - Add the following code in the end of the body on your page

<script src="player/js/player.js"></script>

3 - This is the player structure 

    player/
        ├── css/
        │   ├── animations.css
        │   ├── landscape.css
        │   └── style.css
        └── js/
            ├── img/
            │   └── (Here is where you add the music images)
            ├── musics/
            │   └── (Here is where you add the mp3 files)
            └── player.js
